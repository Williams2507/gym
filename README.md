# GymFlow — React

Protótipo completo de uma plataforma para acompanhar a lotação de academias em tempo real.

## Funcionalidades

- Mapa interativo com OpenStreetMap e marcadores por nível de lotação
- Busca e filtros de academias
- Página completa da academia com fotos, avaliações, estrutura e contato
- Previsão de movimento por horário
- Alertas personalizados
- Favoritos e perfil do aluno
- Painel de gestão da academia
- Simulação automática de atualização da lotação
- Layout responsivo para desktop e celular

## Como executar

```bash
npm install
npm run dev
```

Depois, abra o endereço exibido pelo Vite, normalmente `http://localhost:5173`.

## Build de produção

```bash
npm run build
npm run preview
```

## Observações

Os dados são demonstrativos. Para produção, substitua os dados locais por uma API e integre autenticação, banco de dados, catracas, sensores e notificações push.
